# Tempo Sync 快速入門

5 分鐘內開始使用 Tempo Sync 同步你的工作紀錄到 Jira。

## Step 1: 安裝

```bash
# 推薦使用 uv
uv tool install tempo-sync

# 或使用 pip
pip install tempo-sync
```

## Step 2: 配置 Jira

```bash
tempo setup
```

你需要準備：

| 項目 | 說明 | 如何取得 |
|------|------|----------|
| Jira URL | 你的 Jira 伺服器位址 | 例如 `https://jira.company.com` |
| PAT | Personal Access Token | Jira → 個人設定 → Personal Access Tokens |

執行 `tempo setup` 後依照提示輸入即可。

## Step 3: 配置 LLM（選填）

LLM 會自動彙整你的工作描述，讓 worklog 更簡潔。

```bash
tempo setup-llm
```

**推薦配置（最簡單）：**
- 選擇 `ollama`
- 先安裝 Ollama：`brew install ollama && ollama pull llama3.2`
- 模型填入：`llama3.2`

**或使用 OpenAI：**
- 選擇 `openai`
- 輸入你的 API Key
- 模型填入：`gpt-4o-mini`

## Step 4: 開始使用

```bash
tempo
```

就這麼簡單！程式會：
1. 顯示你的配置狀態
2. 讓你選擇時間範圍（預設過去 7 天）
3. 列出這段時間的工作紀錄
4. 讓你逐日對應 Jira Issue 並上傳

## 常用指令

```bash
# 互動模式（最常用）
tempo

# 快速分析本週
tempo analyze --week

# 分析上週
tempo analyze --last-week

# 分析過去 N 天
tempo analyze --days 14

# 分析後直接上傳
tempo analyze --week --upload

# 查看有紀錄的日期
tempo dates
```
